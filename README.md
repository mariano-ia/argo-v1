# Manual Setup Instructions

Since `npm` was unavailable during generation, you must initialize the project manually before running.

1.  **Install Node.js**: Ensure Node.js (v18+) is installed.
2.  **Install Dependencies**:
    ```bash
    npm install
    # or
    yarn install
    # or
    bun install
    ```
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
